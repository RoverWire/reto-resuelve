# Prueba Ingeniería Resuelve

Ejercicio de backend para Resuelve tu Deuda.

## Requerimientos

- Ruby 2.7.1

## Instalación

Ejecutar en consola:

```bash
bundle install
```

## Desarrollo

Para ejecutar en una máquina local:

```bash
bundle exec rackup
```

Si desea recargar rackup automáticamente al editar un archivo use:

```bash
bundle exec guard
```
